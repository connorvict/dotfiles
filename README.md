# My Chezmoi Dotfiles

Welcome to my personal dotfiles managed by [chezmoi](https://www.chezmoi.io/). This repository contains my configurations for various tools and applications that I use on a daily basis. These configurations are tailored to my personal preferences and workflow, but feel free to use them as a starting point for your own setup.

## Packages

This setup uses a variety of packages to enhance productivity and user experience. Here are some of the key packages:

- **Neovim**: An extendable and highly configurable text editor.
- **fd**: A simple, fast and user-friendly alternative to 'find'.
- **ripgrep**: A line-oriented search tool that recursively searches your current directory for a regex pattern.
- **fzf**: A general-purpose command-line fuzzy finder.
- **zoxide**: A faster way to navigate your filesystem.
- **jq**: A lightweight and flexible command-line JSON processor.
- **yazi**: A blazingly fast terminal file manager
- **keychain**: A manager for ssh-agent, allowing you to have one long-running ssh-agent process per system, rather than per login.
- **bat**: A cat clone with syntax highlighting and Git integration.
- **eza**: A simple, fast and user-friendly alternative to 'ls'.
- **procs**: A modern replacement for 'ps' written in Rust.

## Languages

This setup is optimized for development with the following languages:

- **Go**: An open source programming language that makes it easy to build simple, reliable, and efficient software.
- **Python**: A popular programming language known for its simple syntax and wide range of libraries.
- **Node.js**: A JavaScript runtime built on Chrome's V8 JavaScript engine.

## Installation

To install these dotfiles on a new machine, you can use the following command:

```bash
chezmoi init https://github.com/connorvict/dotfiles.git
```
